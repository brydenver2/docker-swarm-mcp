# Uvicorn

## Overview
An ASGI web server, for Python.

## Short Description
Uvicorn is a lightning-fast ASGI server implementation, using uvloop and httptools. It is the recommended server for running FastAPI applications.

## Key Features
- ASGI server implementation
- High performance with uvloop and httptools
- WebSocket support
- HTTP/2 and HTTP/3 support (with Hypercorn)
- Automatic reloading for development
