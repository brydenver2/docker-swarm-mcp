# FastAPI

## Overview
A modern, fast (high-performance), web framework for building APIs with Python 3.7+ based on standard Python type hints.

## Short Description
FastAPI is a web framework for building APIs with Python that is as fast as NodeJS and Go. It is built on top of Starlette and Pydantic, and provides automatic interactive documentation.

## Key Features
- High performance
- Fast to code
- Automatic interactive documentation (Swagger UI and ReDoc)
- Type hints and data validation with Pydantic
- Dependency Injection system
