# Docker Compose

## Overview
A tool for defining and running multi-container Docker applications using YAML configuration files.

## Short Description
Docker Compose allows you to define your application's services, networks, and volumes in a single YAML file, then spin up your entire application stack with a single command.

## Key Features
- Multi-container application orchestration
- YAML-based configuration
- Service dependency management
- Network and volume management
- Environment variable substitution
- Support for both standalone mode and Swarm mode deployment
